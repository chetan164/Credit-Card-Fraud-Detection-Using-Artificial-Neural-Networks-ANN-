# Credit-Card-Fraud-Detection-Using-Artificial-Neural-Networks-ANN-
🔍 Detected credit card fraud using a deep ANN (TensorFlow/Keras) on 18K+ real-world transactions. Achieved X ROC-AUC and 94% fraud recall despite severe class imbalance (0.44% fraud rate). Pipeline includes EDA, StandardScaler, BatchNorm, Dropout, EarlyStopping &amp; Confusion Matrix evaluation.
